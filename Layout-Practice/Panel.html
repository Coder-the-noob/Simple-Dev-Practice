Click to show panel

This panel contains a div element, which is hidden by default(display:none).

It is styled with CSS and we use Javascript to show it(display:block).

How it works: Notice that hte p element with class="flip" has an onclick attribute to it. When the user clicks on the p element, a function called myFunction() is executed, which changes the style of the div with id="panel" from display:none(hidden) to display:block(visible).

You will learn more about Javascript in our Javascript Tutorial.
